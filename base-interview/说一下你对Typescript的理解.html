# 说一下你对Typescript的理解?

# 核心定位

  • 是 JavaScript 的超集，完全兼容 JS 语法；
  • 为 JavaScript 添加可选的静态类型检查，通过编译时类型校验提升代码可靠性。

# 核心价值

  • 类型安全与错误预防：在编译阶段捕获类型错误，避免运行时崩溃；
  • 提升代码可维护性：通过类型注解明确变量类型。

# 关键特性

  • 类型注解：显示声明变量类型，函数参数类型，返回值类型等；
  • 类型推断：自动推导变量类型；
  • 接口：定义对象结构，函数类型或者类的结构；
  • type：定义类型别名，联合类型，元组类型等；
  • 泛型：定义可复用的组件，支持多种类型；
  • 联合类型：允许一个变量有多个类型；
  • 类型别名：为类型起一个新的名字；

# 适用场景

  • 适用于中大型项目，多人协作，长期维护的项目。
