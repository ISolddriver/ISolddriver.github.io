# vue3 相关总结

# 组合式API

Composition API将某个逻辑代码块封装成函数，然后在需要的地方调用，将逻辑代码块与模板分离，使代码更具可读性，可维护性。

# 生命周期 以 on 开头

  • setup()：在组件创建之前调用，用于初始化响应式数据、计算属性、监听器等，替代了vue2 中的beforeCreated() 和 created()。
  • onBeforeMount()：在组件挂载之前调用，用于执行组件挂载前的准备工作。
  • onMounted()：在组件挂载之后调用，用于执行组件挂载后的操作，如获取数据、操作DOM等。
  • onBeforeUpdate()：在组件更新之前调用，用于执行组件更新前的准备工作。
  • onUpdated()：在组件更新之后调用，用于执行组件更新后的操作。
  • onBeforeUnmount()：在组件卸载之前调用，用于执行组件卸载前的准备工作。
  • onUnmounted()：在组件卸载之后调用，用于执行组件卸载后的操作，如清理定时器、取消订阅等。

# vue3 中为什么没有 this

在 Vue 3 中，setup() 函数是组件的入口函数，它会在组件实例创建之前调用，用于初始化响应式数据、计算属性、监听器等。由于 setup() 函数是在组件实例创建之前调用的，所以此时组件实例还没有被创建，因此无法使用 this 关键字来访问组件实例。 但是可以通过 setup() 函数的参数 ctx 来访问组件实例，ctx 是一个上下文对象，包含了组件实例、props、emit等属性。

setup(ctx, props) {
  // ctx：上下文对象，包含组件实例、props、emit等
  // props：组件的props属性
}

# 全局方法

  1. 在 main.js 中定义全局方法
import { createApp } from 'vue'
import App from './App.vue'

const app = createApp(App)

app.config.globalProperties.$fn = function () {
  console.log('全局方法')
}

app.mount('#app')
  1. 使用插件系统。好处是将全局方法封装起来，更加模块化。
export default {
  install(app) {
    app.config.globalProperties.$fn = function () {
      console.log('全局方法')
    }
  }
}

// 在main.js中引入插件
import { createApp } from 'vue'
import App from './App.vue'
import plugin from './plugin'

const app = createApp(App)
app.use(plugin)
app.mount('#app')

# ref 和 reactive

ref 和 reactive 都是 Vue 3 中用于创建响应式数据的 API，它们的主要区别在于使用方式和适用场景。

ref用于创建一个响应式数据对象。它接受一个参数，可以是基本数据类型或对象，返回一个包含该参数的响应式数据对象。ref 创建的响应式数据对象可以通过 .value 属性来访问和修改。

reactive用于创建一个响应式对象。它接受一个参数，必须是对象，返回一个包含该对象的响应式对象。reactive 创建的响应式对象可以直接访问和修改其属性。

toRef 和 toReactive 是 Vue 3 中用于将普通数据转换为响应式数据的 API。

toValue 用于将 ref 创建的响应式数据对象转换为普通数据。它接受一个 ref 对象作为参数，返回该 ref 对象的值。

# 组件 v-model

从 Vue 3.4 开始，推荐的实现方式是使用 defineModel() 宏

<!-- Child.vue -->
<script setup>
const model = defineModel()

function update() {
  model.value++
}
</script>

<template>
  <div>Parent bound v-model is: {{ model }}</div>
  <button @click="update">Increment</button>
</template>

# 组合式函数 以 use 命名

以组合式API 封装和复用代码，并返回需要暴露的状态，通常以use开头。

export function useCounter() {
  const count = ref(0)
  function update () {
    count.value++
  }
  return {
    count,
    update
  }
}

// 在组件中使用
import { useCounter } from './useCounter'
export default {
  setup() {
    const { count, update } = useCounter()
    return {
      count,
      update
    }
  }
}
