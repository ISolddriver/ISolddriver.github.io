# Set 和 Map

# Set

Set 是一种新的数据结构，它类似于数组，但是成员的值都是唯一的，没有重复的值。

# 特点：

  • 存储唯一的值，不允许重复。
  • 值的类型可以是任意类型，包括对象。
  • 可以遍历，通过 for...of 或者 forEach。

# 常用方法：

  • add(value)：添加一个值，返回 Set 结构本身。
  • delete(value)：删除一个值，返回一个布尔值，表示删除是否成功。
  • has(value)：返回一个布尔值，表示该值是否为 Set 的成员。
  • size：返回 Set 的大小。

# WeakSet

WeakSet 是一种新的数据结构，它类似于 Set，但是成员只能是对象，并且对象的引用不会被垃圾回收机制回收。

# 特点：

  • 存储唯一值，但值必须是对象，
  • 对值的引用是弱引用，不会被垃圾回收机制回收。
  • 不能遍历，没有 forEach 方法，没有size方法。

# 常用方法：

  • add(value)：添加一个值，返回 WeakSet 结构本身。
  • delete(value)：删除一个值，返回一个布尔值，表示删除是否成功。
  • has(value)：返回一个布尔值，表示该值是否为 WeakSet 的成员。
const weakSet = new WeakSet();
const obj1 = {};
const obj2 = {};
weakSet.add(obj1);
weakSet.add(obj2);
console.log(weakSet.has(obj1)); // true

# Map

Map 是一种新的数据结构，它类似于对象，但是键可以是任意类型。

# 特点：

  • 存储键值对，键和值可以是任意类型，包括对象。
  • 键是唯一的，值可以重复。
  • 可以遍历，通过 for...of 或者 forEach。

# 常用方法：

  • set(key, value)：添加一个键值对，返回 Map 结构本身。
  • get(key)：返回键对应的值。
  • delete(key)：删除一个键值对，返回一个布尔值，表示删除是否成功。
  • has(key)：返回一个布尔值，表示该键是否为 Map 的成员。
  • size：返回 Map 的大小。

# WeakMap

WeakMap 是一种新的数据结构，它类似于 Map，但是键只能是对象，并且对象的引用不会被垃圾回收机制回收。

# 特点：

  • 存储键值对，键只能是对象，
  • 对键的引用是弱引用，不会被垃圾回收机制回收。
  • 不能遍历，没有 forEach 方法，没有size方法。

# 常用方法：

  • set(key, value)：添加一个键值对，返回 WeakMap 结构本身。
  • get(key)：返回键对应的值。
  • delete(key)：删除一个键值对，返回一个布尔值，表示删除是否成功。
  • has(key)：返回一个布尔值，表示该键是否为 WeakMap 的成员。
const weakMap = new WeakMap();
const obj1 = {};
weakMap.set(obj1, 'Alice');
console.log(weakMap.get(obj1)); // Alice

# 数组的基本方法

const arr = [1, 2, 3, 4, 5];
arr.push(6); // [1, 2, 3, 4, 5, 6]
arr.pop(); // [1, 2, 3, 4, 5] 删除最后一个元素，并返回删除的值

arr.unshift(0); // [0, 2, 3, 4, 5]
arr.shift(); // [2, 3, 4, 5] 删除第一个元素，并返回删除的值

arr.splice(1, 2); // [2, 3, 4, 5] 从索引1开始删除2个元素
arr.splice(1, 0, 2, 3); // [2, 3, 2, 3, 4, 5] 从索引1开始插入2和3

arr.join(','); // '2,3,4,5' 将数组元素用逗号连接成一个字符串
arr.slice(1, 3); // [3, 4] 返回从索引1到索引3（不包括索引3）的子数组
