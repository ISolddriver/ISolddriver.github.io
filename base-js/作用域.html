# 什么是作用域？

作用域(Scope)是变量与函数的可访问范围。JavaScript采用的是词法作用域，也就是静态作用域，即词法作用域在代码书写时就确定了。

# 作用域的分类

JavaScript的作用域分为全局作用域和局部作用域。

  • 全局作用域：在代码中任何地方都能访问到的对象，在全局作用域中创建的变量可以在代码的任何地方访问。
  • 函数作用域：只在固定的代码片段内可以访问到的对象，最常见的例如函数内部，如果在函数内部没有声明变量或者函数，那么它就是全局变量。
  • 块级作用域：在ES6中新增了let和const关键字，它们可以声明块级作用域的变量。

# 静态作用域

  • 定义：在代码编写时就确定了，而不是在代码运行时确定。
  • 特点：作用域链在函数定义时就已经确定，变量的查找顺序是从当前作用域开始，逐级向上查找，直到全局作用域为止。

# 作用域链

  • 在静态作用域中，每个函数都有一个作用域链，用于变量查找。
  • 作用域链的顺序是：当前函数作用域 -> 外层函数作用域 -> 全局作用域。

# 动态作用域（JavaScript不支持）

  • 定义：在代码运行时确定，而不是在代码编写时确定。
  • JavaScript不支持动态作用域，某些语言如Ruby、PHP等支持动态作用域。
  • 特点：变量的查找顺序基于调用栈，而不是代码的书写顺序。变量的值取决于函数的调用方式，而不是函数的定义方式。
