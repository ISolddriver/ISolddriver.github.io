# 函数柯理化

柯里化是一种将使用多个参数的一个函数转换成一系列使用一个参数的函数的技术。 将一个接收多个参数的函数，拆分为多个步骤，每个步骤接收一个参数，知道所有参数接受完毕，执行原函数。

function curry(fn, ...args) {
  // fn.length 表示函数的参数个数
  return fn.length <= args.length ? fn(...args) : curry.bind(null, fn, ...args)
}
function sum(a, b, c) {
  return a + b + c;
}
const currySum = curry(sum);
console.log(currySum(1)(2)(3)); // 6
console.log(currySum(1, 2)(3)); // 6
console.log(currySum(1)(2, 3)); // 6

  1. 参数复用
  2. 提前返回
