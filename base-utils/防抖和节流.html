# 防抖和节流

# 防抖

在事件被触发n秒后再执行回调，如果在这n秒内又被触发，则重新计时。 应用场景：

  1. 搜索框输入查询
  2. 窗口大小resize
  3. 按钮提交
function debounce(fn, delay, immediate = false) {
  let timer = null;
  return function(...args) {
    const callNow = immediate && !timer;
    clearTimeout(timer);
    timer = setTimeout(() => {
      timer = null;
      if (!immediate) {
        fn.apply(this, args);
      }
    }, delay);
    if (callNow) fn.apply(this, args);
  }
}

# 节流

规定在一个单位时间内，只能触发一次函数。如果这个单位时间内触发多次函数，只有一次生效。 应用场景：

  1. 滚动加载，加载更多或滚到底部监听
  2. 搜索框，搜索联想功能
  3. 高频点击提交，表单重复提交
  4. 计算鼠标移动的距离
  5. canvas绘制动画
  6. DOM 元素的拖拽功能实现（mousemove）
function throttle(fn, delay) {
  let timer = null;
  return function(...args) {
    if (!timer) {
      timer = setTimeout(() => {
        fn.apply(this, args);
        timer = null;
      }, delay);
    }
  };
}
