# 什么是 CSS 层叠上下文?

CSS 的层叠上下文是管理元素层叠顺序的机制，决定了元素在 Z 轴上的显示顺序。

1. 定义

  • 层叠上下文是一个独立的三维空间，其中的元素按照一定的规则在Z轴上排列。
  • 每个层叠上下文都有自己的顺序，并且可以包含子元素的层叠上下文。

2. 形成条件

  • 根元素（HTML）
  • 定位元素：
    • position: absolute; 或 position: relative;且 z-index 不为 auto；
    • position: fixed; 或 position: sticky;。
  • Flex 容器和 Grid 容器：display: flex; 或 display: grid;，且 z-index 不为 auto。
  • 透明度：opacity 小于 1。
  • 变换：transform 不为 none。

3. 层叠顺序规则 在一个层叠上下文中，从下到上的的顺序为：

  • 层叠上下文的背景和边框。
  • z-index值为负的子元素。
  • 块级元素。
  • 浮动元素。
  • 行内元素。
  • z-index值为 0 或者 auto 的定位元素：position 为 absolute、relative 或 fixed，且 z-index 为 auto 或 0。
  • z-index值为正的子元素。

4. 层叠上下文的特性

  • 独立性：
    • 每个层叠上下文都是独立的，内部的层叠顺序不会影响外部。
    • 子元素的z-index只在父层叠上下文中有效。
  • 嵌套性：层叠上下文可以嵌套，子层叠上下文的层叠顺序受父层叠上下文的影响。
  • z-index 的作用范围：
    • z-index只在同一个层叠上下文中有效。
    • 如果两个元素不在同一个层叠上下文中，它们的z-index值不能直接比较。
